1.Introdução

Diagramas de Colaboração são usados para mostrar como objetos diferentes interagem para realizar o comportamento esperado deles em um caso de uso. Diferente do diagrama de sequencia, o diagrama de colaboração mostra os relacionamentos entre os objetos, expessando, assim, informações similares ao diagrama de sequência de maneira diferente.

2.Diagramas

Diagramas de Colaboração representando a interação do usuário com o sistema a fim de visualizar e modificar as informacoes de uma determinada partida.
